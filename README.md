# Graph Container (Варіант 10)

Реалізація класу-контейнера Graph мовою C++ для розрахунково-графічного завдання.

## Проєкт містить три версії програми

### Graph_Basic.cpp
Базова реалізація графа з використанням списків суміжності.

### Graph_Visual.cpp
Розширена версія з консольним графічним відображенням структури графа.

### Virus_Spread_Simulation.cpp
Інтерактивна симуляція поширення вірусу в комп’ютерній мережі на основі графа.

---

# Реалізовані методи

- add_node(T node)
- add_edge(T from, T to)
- remove_node(T node)
- neighbors(T node)

---

# Опис Virus Spread Simulation

Програма моделює поширення комп’ютерного вірусу в мережі, де:
- вершини графа — комп’ютери;
- ребра — мережеві з’єднання між вузлами.

Вірус починає зараження з одного вузла та поступово поширюється через сусідні вузли мережі.

Гравець може захищати окремі вузли, щоб зупинити поширення вірусу та зберегти мережу.

У симуляції реалізовано:
- декілька рівнів складності;
- випадковий шанс зараження;
- статуси вузлів:
  - SAFE
  - INFECTED
  - PROTECTED
- систему перемоги та поразки;
- інтерактивну взаємодію з користувачем.

---

# Правила гри

- Вірус починає зараження з вузла №1.
- Кожного ходу вірус намагається заразити сусідні вузли.
- Гравець обирає один вузол для захисту.
- Захищені вузли не можуть бути заражені.
- Ймовірність зараження залежить від рівня складності.
- Якщо більшість мережі заражена — гравець програє.
- Якщо вдалося захистити достатню кількість вузлів — гравець перемагає.
